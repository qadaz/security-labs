> ## 🧪 Vulnerability Assessment
>
> ### Topics Covered
> - **Reconnaissance** — passive & active information gathering
> - - **Network Scanning** — Nmap, Netdiscover, host enumeration
>   - - **Service Enumeration** — banner grabbing, version detection
>     - - **Vulnerability Scanning** — Nessus, OpenVAS, Nikto
>       - - **Web App Testing** — OWASP Top 10, Burp Suite, SQLi, XSS
>         - - **Exploitation Basics** — Metasploit framework, manual exploitation
>           - - **Post-Exploitation** — privilege escalation, persistence concepts
>             - - **Reporting** — writing clear, actionable security reports
>              
>               - ### Tools Used
>               - | Tool | Purpose |
>               - |------|---------|
>               - | Nmap | Network scanning & enumeration |
>               - | Burp Suite | Web application proxy & testing |
>               - | Nikto | Web server vulnerability scanner |
>               - | Metasploit | Exploitation framework |
>               - | Wireshark | Packet capture & analysis |
>               - | Gobuster | Directory & DNS brute-forcing |
>               - | John the Ripper | Password cracking |
